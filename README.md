## 这些坑，提前知道能少走弯路

用了这么久，也踩过几次坑，总结几条经验：

**IP 一致性很关键**。开卡时用什么地区 IP，消费时就尽量保持同地区 IP。用美国 IP 开的卡突然用印度 IP 消费，被风控的概率会明显上升。

**小额测试再大额**。新卡拿到手先跑一笔 1-2 美元的小额交易，确认通道通畅再做订阅或大额支付，比直接冲 20 美元订阅稳妥。

**Steam、暴雪、EA 这种游戏平台慎用**。这几个平台对虚拟卡的风控特别严，封卡率高，PokePay 官方也没把它们列入推荐场景，真要玩游戏建议用实体卡或者直接走 PayPal。

**CVV 不要频繁查询**。每查一次 CVV 系统都会记录，频繁查会触发风控，需要时查一次记下来就行。

**手续费要算总账**。1% 交易费 + 1% 货币转换费 + 1-2% 跨境费，一笔 100 美元的消费实际到手可能要扣 3-4 美元费用。大额消费前心里要有数。
